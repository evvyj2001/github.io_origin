< !-- 탭메뉴와 탭컨텐츠를 감싸는 태그에 [data-tab-wrap] 속성 추가 -->                            
< div data-tab-wrap> 
    < div tab-menu-wrap> < !-- 탭메뉴를 감싸는 태그에 [tab-menu-wrap] 속성 추가 -->                          
        < button data-tab-menu="all" class="__on">모두 보기< /button> 
        < !-- '모두 보기' 기능을 추가해야하는 경우 해당 탭에 [data-tab-menu="all"] 추가 -->
        < button data-tab-menu="menu1">메뉴1< /button>        
        < !-- 탭메뉴의 각각의 탭에 [data-tab-menu] 속성을 추가하고 
            속성값으로는 연결할 탭컨텐츠의 [data-tab-content]의 속성값과 동일한 값 추가 -->
        < button data-tab-menu="menu2">메뉴2< /button>
        < button data-tab-menu="menu3">메뉴3< /button>
    < /div>
    < div tab-container> < !-- 탭컨텐츠를 감싸는 태그에 [tab-container] 속성 추가 -->
        < div data-tab-content="menu1"> < !-- 각각의 탭컨텐츠마다 [data-tab-content] 
            속성 추가 및 속성 값으로는 연결한 탭메뉴의 [data-tab-menu]값을 부여 -->
            menu1 내용입니다.
        < /div>
        < div data-tab-content="menu2">
            menu2 내용입니다.
        < /div>
        < div data-tab-content="menu3">
            menu3 내용입니다.
        < /div>
    < /div>
< /div>

유의사항

  • 각 항목의 active 상태의 스타일링이 별도로 필요한 경우 __on 클래스를 사용하여 css를 추가해주세요.
    ex) [data-tab-menu].__on { ... }
